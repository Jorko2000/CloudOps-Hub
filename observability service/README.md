# Observability Service

Centralized observability platform for CloudOps Hub.

## Features

* Prometheus Metrics
* OpenTelemetry Tracing
* Grafana Dashboards
* AlertManager Rules
* Structured Logging

## Endpoints

GET /metrics

## Technologies

* Go
* Prometheus
* Grafana
* OpenTelemetry
* Jaeger
* Zap Logger
